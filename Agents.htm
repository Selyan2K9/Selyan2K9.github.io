Agents

Il existe différents types d'agents.

Chaque types d'agents a une particularité,

Les controleurs sont les personnages qui servent a mettre des bombes de fumées pour empechés les ennemis de voir.

Les duelistes sont les personnages qui servent a prendre des duel (d'où le nom) pour faire un maximum d'élimination.

Les initiateurs sont les personnages qui servent a aveugler les ennemis avec toutes sortes d'objets.

Les sentinelles sont les personnages qui servent a piéger le site avec différents piège pour ne pas que les ennemies avance sur le site.
